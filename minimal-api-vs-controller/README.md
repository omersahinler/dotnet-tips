# Minimal API vs Controller — .NET 8+

## Scenario

Building a product API. Two approaches — which one fits your project?

## When to use Minimal API
- Small, focused microservices
- Startup performance matters
- Few endpoints, fast iteration

## When to use Controllers
- Large team, many endpoints
- Complex filter/middleware needs
- Team is new to Minimal API

## Senaryo

Bir ürün API'si kuruyorsunuz. İki yaklaşım — hangisi projenize uyar?

## Minimal API ne zaman
- Küçük, odaklı microservice'ler
- Startup performansı önemliyse
- Az endpoint, hızlı iterasyon

## Controller ne zaman
- Büyük ekip, çok endpoint
- Karmaşık filter/middleware ihtiyacı
- Ekip Minimal API'ye yeni
